# 🛒 Minha Lista de Compras

> Projeto desenvolvido para facilitar o controle financeiro durante as compras no supermercado.

Este aplicativo permite que o usuário defina um limite de gastos e acompanhe, em tempo real, o valor total acumulado à medida que adiciona produtos com suas respectivas quantidades e preços.

## 🚀 Funcionalidades

- **Definição de Orçamento:** Insira quanto deseja gastar antes de começar.
- **Cálculo Automático:** Multiplicação de `quantidade x preço` automática.
- **Saldo em Tempo Real:** Veja quanto ainda resta do seu orçamento.
- **Alerta Visual:** O saldo fica vermelho caso você ultrapasse o limite definido.
- **Compartilhamento:** Botão integrado para enviar a lista final via WhatsApp.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
- **Mobile:** Android Studio com WebView (Java).
- **IDE:** Visual Studio Code e Android Studio.

## 📱 Como rodar o projeto

1. Clone este repositório.
2. Abra a pasta no **Android Studio**.
3. Conecte seu dispositivo Android (com modo desenvolvedor ativo).
4. Clique em `Run` para instalar o APK.
